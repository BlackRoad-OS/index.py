## Overview

**Central Repository Index for BlackRoad OS** - The comprehensive catalog and discovery service for all repositories in the BlackRoad-OS organization.

Part of the **BlackRoad OS** ecosystem - The Road to AI Sovereignty 🛣️

This repository serves as the authoritative index of all BlackRoad OS projects, providing:
- 📚 **Repository Catalog** - Complete listing of all BlackRoad OS repositories
- 🔍 **Discovery Service** - Find repositories by category, technology, or purpose
- 📊 **Metadata Management** - Centralized repository metadata and relationships
- 🔗 **Integration Hub** - Connection point for all BlackRoad OS services
- 📈 **Statistics & Metrics** - Organization-wide repository analytics

## Quick Start

```bash
# Clone this repository
git clone https://github.com/BlackRoad-OS/index.py.git
cd index.py

# Install dependencies
pip install -r requirements.txt

# Run the index generator
python index.py

# Generate HTML catalog
python index.py --format html --output catalog.html

# Query specific categories
python index.py --category infrastructure
python index.py --language Python
```

## Usage

### Python API

```python
from blackroad_index import RepositoryIndex

# Initialize the index
index = RepositoryIndex()

# Fetch all repositories
repos = index.fetch_all()

# Filter by category
ai_repos = index.filter_by_category('ai')

# Filter by language
python_repos = index.filter_by_language('Python')

# Get repository metadata
metadata = index.get_metadata('roadscheduler')

# Generate reports
index.generate_report(format='html', output='report.html')
```

### Command Line

```bash
# List all repositories
python index.py --list

# Filter by category
python index.py --category infrastructure

# Filter by language
python index.py --language Python

# Generate full catalog
python index.py --catalog

# Export as JSON
python index.py --export repos.json

# Update repository cache
python index.py --update

# Show statistics
python index.py --stats
```

## Configuration

Create a `config.yaml` file to customize the index behavior:

```yaml
organization: BlackRoad-OS
cache_ttl: 3600
output_formats:
  - json
  - html
  - markdown
categories:
  - core
  - ai
  - devops
  - web
  - media
  - business
  - specialized
  - blockchain
api_endpoints:
  github: https://api.github.com
  docs: https://docs.blackroad.io
  support: https://support.blackroad.io
```
